HTML

HTML (Lenguaje de marcado de hipertexto o HyperText Markup Language por sus siglas en inglés) es un lenguaje descriptivo que especifica la estructura de las páginas web

Historia de HTML

El origen de HTML se remonta a 1980, cuando el físico Tim Berners-Lee, investigador del CERN (Organización Europea para la Investigación Nuclear) propuso un nuevo sistema de “hipertexto” para compartir documentos. Los sistemas de hipertexto habían sido desarrollados años antes.

Versiones de HTML

Tim Berners-Lee definio la primera version de HTML en el año 1991

En la actualidad la version de HTML es HTML5
